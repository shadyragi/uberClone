# uberClone is a real-time clone for uber app written with firebase where there is two types of users <br>
1 - user who can: <br>
a - Search for places <br>
b - request a ride <br>
c - login/logout <br>

2 - driver who can: <br>
a - Accept a trip <br>
b - Reject a trip <br>
c - login/out
